# 📖 About

**Sadness World** is a browser-based emotional content platform created to provide users with a calm and immersive digital experience.

The website brings together emotional writings, carefully selected music, videos, inspiring quotes, sad biography ideas, and a unique **Daily Mood Prediction** feature within a modern and visually appealing interface.

Rather than focusing on a single type of content, Sadness World combines multiple forms of emotional media to create an engaging and relaxing experience.

---

# ✨ Features

* 🌧️ Emotional writings
* 🎵 Music collection
* 🎥 Emotional videos
* 💭 Inspirational & emotional quotes
* 📝 Sad biography ideas
* 🔮 Daily Mood Prediction
* 🎨 Modern and attractive user interface
* 📱 Responsive design
* ⚡ Lightweight and fast

---

# 🎯 Project Goals

The goal of Sadness World is to create a visually appealing platform where users can explore emotional content in different formats while enjoying a smooth and engaging browsing experience.
